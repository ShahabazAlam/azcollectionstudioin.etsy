# Privacy Policy

**AZ Collection – AI Content Generator**

Last updated: 2026

## 1. Introduction

AZ Collection operates an AI-powered tool that helps creators and online sellers generate optimized visual content for social platforms such as Pinterest and Instagram.

This Privacy Policy explains how we collect, use, and protect user data when interacting with our application.

---

## 2. Information We Collect

Our application may collect the following limited information:

* User authentication data from Pinterest or Instagram when the user connects their account
* Basic account identifiers required for publishing content
* Product images uploaded by users for content generation
* Generated content such as captions, descriptions, and tags

We do **not collect sensitive personal information** such as passwords, payment details, or private messages.

---

## 3. How We Use Information

Collected information is used only for the following purposes:

* Generating Pinterest Pins and Instagram posts
* Publishing content to a user's connected social media account
* Improving content generation features
* Ensuring proper functionality of the application

Users review and approve generated content before publishing.

---

## 4. Third-Party Services

Our application may interact with third-party APIs including:

* Pinterest API
* Instagram Graph API
* AI processing services for image and text generation

These services operate under their own privacy policies.

---

## 5. Data Storage

We store minimal data required to operate the service, including:

* temporary generated content
* image processing results
* authentication tokens required for API access

Data is stored securely and access is restricted.

---

## 6. Data Sharing

We do **not sell, rent, or share user data with third parties** for marketing purposes.

Data is only shared when required to interact with official APIs such as Pinterest or Instagram for publishing content requested by the user.

---

## 7. User Control

Users have full control over their content and may:

* disconnect their social media accounts at any time
* delete generated content
* stop using the application

---

## 8. Security

We implement reasonable security measures to protect user data from unauthorized access or misuse.

However, no internet service can guarantee absolute security.

---

## 9. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a revised date.
